🧬 TabClaw 如何越用越聪明
了解自进化流程，让 TabClaw 成为你的专属数据分析师
1
📊 上传表格，提出问题
上传 CSV 或 Excel 文件，用自然语言描述你想要的分析。TabClaw 会制定计划、调用工具、给出结论。
↓
2
👍👎 给出反馈
每次分析完成后，点击 👍 或 👎 告诉 TabClaw 效果如何。这是它学习的核心信号——不满意的结果会驱动改进。
↓
3
🧠 自动学习技能
开启底部的 Skill Learning 开关后，TabClaw 会从复杂任务中自动提炼可复用的分析技能。你也可以点击 Skills 面板的 🔍 按钮，主动扫描历史记录发现重复模式并采纳为技能。
↓
4
⬆ 反馈驱动进化
当某个技能累计收到 2 次以上 👎 时，TabClaw 会自动分析失败原因并升级该技能。技能版本号递增，旧版本保留。你也可以在技能详情里手动点 ⬆ Improve 触发改进。
↓
5
📈 越来越好
下次问类似问题时，进化后的技能会自动注入 Agent 的思维链，让分析更准确、更有条理。点击侧边栏 成长报告 查看领域熟练度、效率变化和里程碑。
💡 快速上手建议
  • 多给反馈——👍👎 是 TabClaw 进化的燃料
  • 开启 Skill Learning——让系统自动从每次分析中学习
  • 定期点 🔍 发现技能——从历史中挖掘你还没注意到的分析模式
  • 查看成长报告——观察 TabClaw 在各个领域的进步
